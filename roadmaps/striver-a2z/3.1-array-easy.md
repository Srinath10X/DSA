# Array Easy Questions 

## 1.Largest Element in an Array
```cpp
#include <bits/stdc++.h>

int main() {
  int arr[] = {1, 2, 3, 4, 5};

  int max = arr[0];
  for (int i = 0; i < std::size(arr); i++) {
    if (arr[i] > max) {
      max = arr[i];
    }
  }

  std::cout << max << std::endl;

  return 0;
}
```
